# Document Templates

## Beacon (Long-Form Analysis)

**Purpose:** Deep-dive analytical piece synthesizing macro dynamics, monetary mechanics, and market technicals.

**Structure:**
```
[Banner or Logo]
[Title - Montserrat Bold 28pt]
[Subtitle/Date - Inter 14pt, Ocean Blue]
[Accent Bar]

[Executive Summary - 2-3 paragraphs]

[Section 1: Macro Context]
  - Key data points
  - Charts (1-2)
  - Analysis

[Section 2: Monetary/Liquidity Dynamics]
  - Fed operations
  - Charts (2-3)
  - Implications

[Section 3: Market Technical Setup]
  - Cross-asset signals
  - Charts (2-3)
  - Positioning considerations

[Conclusion/Outlook]

[Sign-off: "That's our view from The Watch. Until next time, we'll be sure to keep the light on..."]

[Footer with contact info]
```

**Format:** PDF or HTML
**Recommended layout:** Portrait, single-column for readability
**Chart treatment:** Full-width, captioned, Ocean Blue border

## Beam (Single-Chart Insight)

**Purpose:** Quick-hit insight centered on one compelling chart.

**Structure:**
```
[Logo - small, top-left]

[Chart Title - Montserrat Bold 18pt]
[Date - Inter 10pt]

[CHART - full width, bordered]
[Caption]

[Commentary - 150-300 words max]
  - What it shows
  - Why it matters
  - Implication

[Footer]
```

**Format:** HTML (for Substack), PDF, or social-optimized image
**Recommended layout:** Portrait or square
**Chart treatment:** Full-width, prominent, captioned

## Chartbook (Visual Compilation)

**Purpose:** Comprehensive visual diagnostic of macro landscapeâ€”50-75 institutional-quality charts.

**Structure:**
```
[Cover Page]
  - Banner
  - "CHARTBOOK"
  - Date/Issue Number
  - Accent Bar

[Table of Contents - optional for 50+ charts]

[Section Dividers with Section Title]

[Chart Pages]
  - 2-4 charts per page (landscape)
  - Or 1-2 charts per page (portrait)
  - Consistent numbering: Chart 1, Chart 2...
  - Brief caption for each

[Sections typically include:]
  1. Macro Overview (GDP, inflation, employment)
  2. Labor Market Dynamics (LFI/LDI framework)
  3. Monetary & Liquidity (Fed balance sheet, RRP, reserves)
  4. Treasury Market
  5. Credit & Spreads
  6. Equity Market Internals
  7. Cross-Asset & Relative Value
  8. Global/EM Dynamics

[Closing Page]
  - Sign-off
  - Contact info
  - "Join The Watch" CTA
```

**Format:** PDF (primary), PPTX (for presentations)
**Recommended layout:** Landscape for multi-chart pages
**Chart treatment:** Bordered, captioned, consistently sized within each page

## Horizon (Forward Outlook)

**Purpose:** Strategic forward-looking piece synthesizing current state into actionable outlook.

**Structure:**
```
[Banner]
[Title: "HORIZON: [Month Year]"]
[Subtitle - Inter 14pt]
[Accent Bar]

[State of Play - current positioning summary]

[Key Themes for Coming Month]
  - Theme 1 with supporting chart
  - Theme 2 with supporting chart
  - Theme 3 with supporting chart

[Risk Matrix]
  - Upside scenarios
  - Downside scenarios
  - Base case probabilities

[Watchlist]
  - Data releases
  - Fed communications
  - Technical levels

[Tactical Implications]

[Sign-off]
[Footer]
```

**Format:** PDF
**Recommended layout:** Portrait
**Chart treatment:** Integrated with narrative, captioned

## Framework Documents

**Purpose:** Methodology explanations (Three Pillars, MRI, LCI, etc.)

**Structure:**
```
[Banner]
[Framework Name - Montserrat Bold 28pt]
[Version/Date]
[Accent Bar]

[Overview - what it is, why it matters]

[Components]
  - Component 1: explanation + formula if applicable
  - Component 2: explanation + formula
  - ...

[Visualization]
  - Example chart showing framework in action

[Interpretation Guide]
  - What high/low readings mean
  - Historical context

[Update Frequency/Data Sources]

[Footer]
```

**Format:** PDF, HTML
**Chart treatment:** Annotated examples showing framework application

## Presentation Slides (PPTX)

**Slide Templates:**

**Title Slide:**
```
[Banner - centered or top]
[Presentation Title - Montserrat Bold 36pt]
[Subtitle/Date - Inter 18pt]
[Accent Bar at bottom]
```

**Section Divider:**
```
[Ocean Blue background]
[Section Title - Montserrat Bold 32pt, White]
[Lighthouse icon - bottom right, subtle]
```

**Content Slide:**
```
[Logo - top left, small]
[Slide Title - Montserrat Bold 24pt, top]
[Content area - bullets or chart]
[Page number - bottom right]
```

**Chart Slide:**
```
[Logo - top left]
[Chart Title - Montserrat Bold 20pt]
[Chart - centered, bordered, 70-80% of slide area]
[Caption - Inter 12pt, below chart]
[Page number]
```

**Closing Slide:**
```
[Banner - centered]
[Contact info block]
["Join The Watch" CTA]
[Accent Bar]
```

## One-Pagers / Informational

**Purpose:** Single-page summaries for quick reference or client materials.

**Structure:**
```
[Logo - top left]
[Title - top right or centered]
[Accent Bar]

[Key Message / Headline - prominent]

[3-4 Key Points with icons or bullets]

[Supporting Chart or Visual - if applicable]

[Contact strip at bottom]
```

**Format:** PDF, DOCX
**Layout:** Portrait, single page
**Design:** Clean, scannable, high information density
